Samsung Galaxy Core Plus SM-G350 (_cs02_)
======================

W.I.P Tree. Almost ready for daily use!

### What does work
  - EGL
  - Audio
  - Wifi
  - RIL
  - Storages
  - Vibration
  - Buttons
  - Earjack
  - Camera (Partially)
  - MTP

### What doesn't work
  - NFC
  - Bluetooth (partially)
