Samsung Galaxy Core Plus SM-G350 (_cs02_)
======================

W.I.P Tree. Almost ready for daily use!

### What does work
  - EGL
  - Wifi
  - Bluetooth
  - RIL
  - Storages
  - Vibration
  - Buttons
  - Earjack
  - Camera (Partially)
  - MTP

### What doesn't work
  - Audio outputs
  - Microphone
  - NFC (not tested/installed)
